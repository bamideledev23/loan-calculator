# Pro Loan Calculator (Python CLI)

A professional command-line loan calculator built in Python, designed to calculate monthly payments, total interest, and total repayment for different loan types. Ideal for personal, car, or mortgage loans. Can be customized to any currency (default: Nigerian Naira ₦).

## Features

- Menu system to select loan type: personal, car, mortgage
- Enter loan amount, annual interest rate, and loan duration in months
- Choose interest type: flat or reducing balance
- Optional extra monthly payments
- Calculates monthly payment, total interest, and total repayment
- Results displayed in Nigerian Naira (₦), easily customizable
- Error handling for invalid input
- Option to calculate another loan or exit
- Optional: save calculation to a text file for records
